# Admin Starter

Golang 管理系统启动器。其他语言版本：
- Ruby 版：[Admin Template](https://github.com/songhuangcn/admin-template)

## 功能

- Makefile 项目管理
- 登录认证
- RBAC 权限管理，权限由路由表自动生成
- I18n 多语言支持

## 技术栈

- Gin
- Vue.js
- MySQL
- Docker
- GitLab CI
- Kubernetes
